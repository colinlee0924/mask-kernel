# Data Analysis Skill

This skill enables you to analyze and visualize data.

## Capabilities

1. **Load Data**: Load data from CSV, JSON, or Excel files
2. **Statistics**: Calculate descriptive statistics
3. **Filtering**: Filter and query datasets
4. **Aggregation**: Group and aggregate data
5. **Visualization**: Generate charts and plots

## Usage Guidelines

- Always inspect the data structure before analysis
- For large datasets, start with summary statistics
- When visualizing, consider the most appropriate chart type
- Provide clear labels and titles for visualizations

## Example Interactions

User: "Show me summary statistics for sales.csv"
- Load the CSV and calculate mean, median, std for numeric columns

User: "What are the top 10 products by revenue?"
- Load data, sort by revenue column, return top 10 rows

User: "Create a bar chart of monthly sales"
- Aggregate data by month and generate visualization
